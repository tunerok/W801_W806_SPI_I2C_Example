# W801_W806_SPI_I2C_Example
W801_W806_SPI_I2C_Example


**Docs** contains the documentation for the chip.

**CKLink** folder contains the firmware of the programmer.

**Code** contains an example of using HAL SPI and I2C for W806 and W801 chips.
